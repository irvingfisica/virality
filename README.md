# data_analysis_template
El repo es un template para armar proyectos de análisis de datos.

## Estructura
- El repositorio tiene la siguiente estructura:

```.
├── LICENSE
├── README.md
├── .gitignore     
├── data 
│   ├── originales
│   ├── procesados           
│   └── consumibles
├── geodata 
│   ├── originales
│   ├── procesados           
│   └── consumibles
├── imagenes 
│   ├── procesados           
│   └── consumibles
├── mapas 
│   ├── procesados           
│   └── consumibles
├── qgis 
│   ├── proyectos           
│   └── estilos  
├── documentos 
│   ├── insumos
│   ├── reportes           
│   └── presentaciones          
└── codigos                 
 
```
